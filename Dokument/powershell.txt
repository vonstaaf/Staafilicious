Starta projektet med rensad cache
npx expo start -c

- Starta projektet normalt
npx expo start

- Kör i webbläsaren
Tryck w i terminalen när Metro bundler kör.
- Kör i Android‑emulator eller Expo Go
Tryck a.
- Kör i iOS‑simulator (Mac)
Tryck i.

- Installera rätt versioner för din SDK
npx expo install

- Lägg till ett nytt paket (exempel: react-native-reanimated)
npx expo install react-native-reanimated


- Bygg för Android
npx expo run:android

- Bygg för iOS (Mac krävs)
npx expo run:ios


- Publicera en uppdatering (OTA)
npx expo publish
- Bygg för distribution med EAS (rekommenderat)
npx expo install eas-cli
npx eas build -p android
npx eas build -p ios
